🏠 House Price Prediction & Regression Analysis

A complete machine learning regression project using the Kaggle House Prices – Advanced Regression Techniques dataset.

This project compares Linear, Ridge, and Lasso Regression models, performs cross-validation, and provides detailed visual analysis including residual plots and distribution comparisons.

📌 Project Objective

To predict house sale prices using selected numerical features and evaluate multiple regression techniques for performance and generalization.

The target variable (SalePrice) is log-transformed to reduce skewness and improve regression stability.

📊 Features Used
	•	GrLivArea – Above ground living area
	•	BedroomAbvGr – Number of bedrooms
	•	FullBath – Number of bathrooms
	•	OverallQual – Overall material & finish quality
	•	GarageCars – Garage capacity
	•	TotalBsmtSF – Basement area

Target:
	•	SalePrice (log-transformed)
  
🧠 Machine Learning Techniques
	•	Linear Regression
	•	Ridge Regression (L2 Regularization)
	•	Lasso Regression (L1 Regularization)
	•	StandardScaler for feature normalization
	•	5-Fold Cross Validation
	•	Log transformation of target variable
	•	Model comparison using R² and MSE
